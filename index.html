Security Architecture

Governance That Cannot Be Bypassed

Dual-layer enforcement: the application layer handles business rules; Turnkey's hardware TEE enforces hard limits even if Tutelum's servers are compromised.

Application Layer
✓Complex business rules & cross-agent budgets
✓Behavioral anomaly detection (async, zero latency)
✓Human approval workflows via Slack / email / webhook
✓Agent identity & risk scoring at scale
✓Payment method routing (x402, USDC, card, ACH)
Turnkey TEE Hardware Layer
🔒Hard spend limits enforced in hardware silicon
🔒Destination allowlists at the key-signing level
🔒Multi-party approval quorums for large transfers
🔒Cryptographic proof of every signing event
🔒Survives a fully compromised application server
CISO Answer: Can an agent spend outside policy if Tutelum's servers are breached? No. The hard spend floor is enforced in hardware by Turnkey's secure enclave: cryptographically verifiable.
